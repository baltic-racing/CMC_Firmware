In diesem Projekt fin det sich die Firmware für die uFAR die im TY21 verbaut ist. Sie fasst die CMC und die FAR aus dem TY19 zusammen und erweitert die Features liste um Sachen wie Fuse Readout.

Sie basiert maßgeblich auf der TY19 CMC Firmware

Geschrieben mit Microchip Studio in C für den AT90CAN128
